QueryEntitiesCursorRequest

Home > @backstage/catalog-client > QueryEntitiesCursorRequest

A request type for CatalogClient.queryEntities(). The method takes this type in a pagination request, following the initial request.

Signature:
type QueryEntitiesCursorRequest = {
    fields?: string[];
    limit?: number;
    cursor: string;
};
